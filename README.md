# 🎵 Song Lyrics Finder

A simple web app that fetches and displays song lyrics using song and artist name.

## 🚀 Features
*  Search lyrics by song name and artist
  
*  Clean and formatted lyrics display
  
*  Removes timestamps from lyrics
  
*  Scroll-based progress bar
  
*  Loading spinner animation
  
*  Input validation and error handling
  
*  Responsive UI design

## 🛠️ Technologies Used
*  HTML5 : Structure of the application
  
*  CSS3 : Styling and animations
  
*  Tailwind CSS (CDN) : UI design and layout
  
*  JavaScript (Vanilla JS) : Logic and API handling
  
*  Fetch API : To retrieve lyrics data
  
*  LRCLIB API : Source of song lyrics


## ⚙️ How It Works
*  User enters song name and artist name
  
*  App sends a request to the lyrics API
  
*  Lyrics are fetched and cleaned (timestamps removed)
  
*  Displayed line-by-line with hover effects
  
*  Progress bar updates as user scrolls

##❗ Validation & Error Handling
*  Prevents empty input submission
  
*  Limits input length (max 100 characters)
  
*  Handles network and API errors gracefully
  
*  Shows user-friendly error messages

## 🎨 UI Highlights

*  Gradient animated background
  
*  Smooth hover effects on lyrics
  
*  Custom loading spinner
  
*  Scroll progress indicator

### 📌 Note
This project uses only frontend technologies and a public API—no backend involved.
